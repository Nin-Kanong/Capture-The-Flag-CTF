### Combine the File Parts
To reassemble the split files, use the ``cat`` command with a wildcard (``*``) to read them in alphabetical order, and redirect the output (``>``) into a new file named ``combined.zip``:
````
cat part_aa part_ab part_ac part_ad part_ae > flag.zip
````

## Quick Reference: Linux Commands Used
| Command / Symbol | What It Does                                | Example                     | Explanation                                    |
| ---------------- | ------------------------------------------- | --------------------------- | ---------------------------------------------- |
| `ssh`            | Connect securely to another computer/server | `ssh user@host -p 22`       | Connect to remote machine using SSH on port 22 |
| `ls`             | Show files and folders                      | `ls -la`                    | `-l` = detailed list, `-a` = show hidden files |
| `cat`            | Display file contents                       | `cat file.txt`              | Prints text inside file                        |
| `*` (Wildcard)   | Matches many files                          | `part_*`                    | Matches `part_a`, `part_b`, etc.               |
| `>`              | Save output into a file                     | `cat a.txt b.txt > all.txt` | Combines files into `all.txt`                  |
| `file`           | Detect file type                            | `file test.zip`             | Shows whether file is ZIP, text, image, etc.   |
| `unzip`          | Extract ZIP files                           | `unzip archive.zip`         | Extracts ZIP contents                          |
