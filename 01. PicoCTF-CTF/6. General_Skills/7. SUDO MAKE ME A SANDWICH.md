### Enumerate Sudo Privileges
> The hint asks: ``How do you know what permission you have?``

In Linux, the command to list what commands a user is allowed to run as root (via ``sudo``) is ``sudo -l``. Run it now:
````
sudo -l
````
<img width="1443" height="173" alt="image" src="https://github.com/user-attachments/assets/e9cc3230-0812-4c52-a68e-5c80f5900c4e" />

The output reveals that the ``ctf-player`` user can run ``/bin/emacs`` as the ``root`` user without needing a password (``NOPASSWD``). 



### Exploit Emacs to Read the Flag
If ywe try to read the flag normally, you will get a "Permission denied" error:
````
ls
cat flag.txt
````
<img width="474" height="145" alt="image" src="https://github.com/user-attachments/assets/4c76a56b-4868-4b67-bbfd-6bfdf04f0293" />


However, because we can run ``emacs`` as root, we can use the text editor to open and read the restricted file! 


Run the following command to open the flag file using root privileges:
````
sudo emacs -nw flag.txt
````
<img width="768" height="276" alt="image" src="https://github.com/user-attachments/assets/f35b595c-b0a7-45b5-9634-d73c8393507c" />

> Once inside Emacs, press ``Esc``, then ``x`` (or ``Alt+X``), type ``shell``, and press ``Enter``.

Now we got flag.


---
> Why ``-nw``?
> The ``-nw`` flag stands for "no window". It forces Emacs to run in text-only mode inside your terminal. If you omit this, Emacs will try to open a graphical window, which will fail over an SSH connection.

### Quick Reference: Linux Commands Used
| Command      | Purpose                           | Example                   | Explanation                                         |
| ------------ | --------------------------------- | ------------------------- | --------------------------------------------------- |
| `ssh`        | Securely connect to remote server | `ssh user@host -p 49846`  | Connects to server using SSH on custom port `49846` |
| `sudo -l`    | Show allowed sudo permissions     | `sudo -l`                 | Lists commands the current user can run as root     |
| `sudo <cmd>` | Run command as administrator/root | `sudo emacs -nw flag.txt` | Opens `flag.txt` with root privileges               |
| `emacs -nw`  | Open Emacs in terminal mode       | `emacs -nw file.txt`      | Opens text editor inside terminal (no GUI)          |
