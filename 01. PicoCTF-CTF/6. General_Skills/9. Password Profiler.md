### Challenge Overview

This challenge teaches **Targeted Password Attacks**. Instead of trying every possible password in the world (Brute Force), we use OSINT (Open Source Intelligence) to build a custom dictionary of passwords based on the target's personal life (names, birthdays, pets).

- ``userinfo``: Contains the target's personal details (First name, last name, birthdate, pet names, etc.). Read this carefully and write the details down.
- ``hash``: Contains the SHA-1 hash of the password we need to crack.
- ``check_password``: A script provided by the challenge to verify if a password is correct. (Usually, we run it like ``python3 check_password "our_guess"``).

### Install CUPP (Common User Passwords Profiler)

**CUPP** is a tool that asks you questions about a target and generates a highly customized wordlist. It is not installed by default, so we need to download it from GitHub.

<img width="1919" height="1025" alt="image" src="https://github.com/user-attachments/assets/70626669-f16b-425d-8f6f-1d922a7ede60" />
