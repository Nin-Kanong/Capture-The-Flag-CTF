````
if user_input == "\x65\x65\x65":
    print(open("./flag.txt", "r").read())
````
- ``\x65`` is hexadecimal notation for the byte value ``0x65``
- ``0x65`` in decimal = ``101``
- ASCII decimal ``101`` = character ``'e'`` 

So: ``"\x65\x65\x65"`` == ``"eee"``
