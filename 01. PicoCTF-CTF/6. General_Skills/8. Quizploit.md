### Analyze the Source 


Read the C source code to understand the vulnerability:


Back to our `vuln.c`:
````
cat vuln.c
````
<img width="834" height="630" alt="image" src="https://github.com/user-attachments/assets/e16126bf-47f6-4ebf-af05-01c570b3837a" />


Look closely at the ``vuln()`` function. It will look something like this:
````
void vuln() {
    char buffer[0x15];          // <--- Look at this line
    fgets(buffer, 0x90, stdin); // <--- And this line
    // ...
}
````
Also, look at the rest of the file to see which functions are called in ``main()``. We will see ``vuln()`` is called, but another function is defined and never used.
