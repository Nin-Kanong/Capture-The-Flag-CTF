### Encoding & Decoding Recognition Table
| Encoding/Cipher | How to Recognize                | Character Set   | Example              | Tool/Command      |
| --------------- | ------------------------------- | --------------- | -------------------- | ----------------- |
| Base64          | Ends with `=` or `==`           | A-Z a-z 0-9 + / | `SGVsbG8=`           | `base64 -d`       |
| Base32          | Mostly uppercase and digits 2-7 | A-Z 2-7         | `JBSWY3DP`           | CyberChef         |
| Base58          | No 0,O,I,l                      | A-Z a-z 1-9     | Bitcoin-like strings | CyberChef         |
| Base85          | Lots of symbols                 | Mixed           | `<~87cURD~>`         | CyberChef         |
| Hex             | Only 0-9,a-f                    | Hex chars       | `48656c6c6f`         | `xxd -r -p`       |
| URL Encode      | Contains `%xx`                  | `%20,%3A`       | `%7Bflag%7D`         | URL Decode        |
| HTML Encode     | `&lt; &gt;`                     | HTML entities   | `&lt;h1&gt;`         | HTML Decode       |
| Binary          | Only 0 and 1                    | Binary          | `01001000`           | Binary Decode     |
| Octal           | Digits 0-7                      | Octal           | `110 145`            | Octal Decode      |
| Decimal ASCII   | Decimal numbers                 | 0-255           | `72 101 108`         | ASCII Decode      |
| Unicode         | `\uXXXX`                        | Unicode         | `\u0048`             | Unicode Decode    |
| ROT13           | Readable gibberish              | Letters         | `uryyb`              | ROT13             |
| Caesar          | Shifted alphabet                | Letters         | `khoor`              | Brute Force       |
| Vigenère        | Alphabetic ciphertext           | Letters         | `LXFOPV`             | Need key          |
| XOR             | Random-looking                  | Any             | Hex output           | XOR Analyzer      |
| Morse           | Dots/dashes                     | . -             | `.... .`             | Morse Decode      |
| Bacon           | A/B pattern                     | A,B             | `AABBA`              | Bacon Decode      |
| Rail Fence      | Scrambled words                 | Letters         | Various              | Rail Fence Decode |
| Affine          | Substitution                    | Letters         | Various              | Affine Solver     |
| Atbash          | Reversed alphabet               | Letters         | `svool`              | Atbash            |
| Playfair        | Digraph cipher                  | Letters         | Various              | Playfair Solver   |
