| Transformation | Command to Reverse           | Description                      |
| -------------- | ---------------------------- | -------------------------------- |
| Base64 encode  | `base64 -d`                  | Decodes Base64 string            |
| Reverse text   | `rev`                        | Reverses character order         |
| `_ → -`        | `tr '-' '_'`                 | Replaces dashes with underscores |
| `{} → ()`      | `tr '()' '{}'`               | Replaces parentheses with braces |
| ROT13          | `tr 'A-Za-z' 'N-ZA-Mn-za-m'` | Reverses ROT13 cipher            |





### Tip:
- **Read hints carefully** - They tell you exactly what transformation was applied
- **Reverse in opposite order** - If transformations were applied A→B→C, reverse them C→B→A
- **Test commands locally** - You can test each command on your machine before submitting
- **Understand tr syntax** - tr 'SET1' 'SET2' maps each character in SET1 to corresponding character in SET2
